# TASTE SAMV71 RTEMS Runtime tests
This is a small suite of tests aimed at exercising the hardware and software independent part of the SAMV71 runtime code. The full runtime is integration tested in this repository: [TASTE SAMV71 RTEMS Tests](https://github.com/n7space/TASTE-SAMV71-RTEMS-Tests).

In order to execute the tests, run:
```
make test
```
